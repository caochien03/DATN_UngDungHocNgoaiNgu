
  # Language Learning Website

  A modern web application for practicing and improving language skills through interactive exercises.

  ## Tech stack

  - React 18
  - Vite
  - TypeScript
  - Tailwind CSS
  - Radix UI / custom UI components

  ## Getting started

  1. Install dependencies:

     ```bash
     npm install
     ```

  2. Start the development server:

     ```bash
     npm run dev
     ```

  3. Open the URL printed in the terminal in your browser.

  ## Project goals

  - Provide an engaging interface for learning vocabulary, grammar, and speaking.
  - Offer multiple practice modes such as quizzes, flashcards, and matching games.
  - Make it easy to extend with new lessons and exercises over time.
